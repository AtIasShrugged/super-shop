## Description

E-shop Project on Nestjs

## Installation

```bash
$ yarn install
```

## Setup

Setup postgresql and fill out .env file

## Seed db

```bash
yarn prisma db seed
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
